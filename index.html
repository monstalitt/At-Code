Looking at Code: Learn HTML Basics

Welcome to the Wonderful World of HTML!

This website is your guide to understanding HTML, the building blocks of web pages. HTML stands for HyperText Markup Language, and it defines the structure and content of web pages. Let's start with the basic building blocks: elements and tags.

An element is a specific part of a web page, like a heading, paragraph, image, or list. Elements are defined using tags, which are special keywords placed around the content. Here's an example:

<h1>This is a heading</h1>

In this example, `<h1>` is the opening tag that defines a heading element (h1), and `</h1>` is the closing tag that marks the end of the element. The text between the tags ("This is a heading") is the content of the element.

Elements with Different Functions

There are many different HTML elements, each serving a specific purpose. Let's explore some common ones:

  • Headings (h1-h6): Used to structure page content with different levels of importance.
  • Paragraphs (<p>): Define blocks of text content.
  • Images (<img>): Embed images within the page.
  • Lists (<ul> and <ol>): Create unordered (bullet points) or ordered (numbered) lists.
  • Links (<a>): Create hyperlinks to other web pages or sections of the same page.

By combining these elements, you can create the structure and content of a web page.
